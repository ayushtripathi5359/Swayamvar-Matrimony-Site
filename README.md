# Swayamvar Matrimony Site

## Overview
This repository contains a React frontend and a Node.js + Express backend for the Swayamvar Matrimony experience.

## Frontend Setup
```bash
npm install
cp .env.example .env
npm run dev
```

The frontend expects the backend to be running on `VITE_API_URL`.

## Backend Setup
```bash
cd backend
npm install
cp .env.example .env
npm run dev
```

See `backend/README.md` for API documentation and auth flow details.
